# Examples

This folder contains practical examples for `Visual Editor Pro`.

## Contents

- `before-after.md` — a short explanation of the original image and the polished result.
- `original.jpeg` — the original source image.
- `polished.jpeg` — the edited version produced with the skill.

## Purpose

These examples show how the skill turns a raw image into a more controlled editorial result while keeping the image believable and grounded.
