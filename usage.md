# Usage Guide

`Visual Editor Pro` is a skill for structured photo diagnosis, retouching recommendations, and editorial-style visual direction.

## Input

Provide a source image, ideally an original photo from a real photography workflow.

## Output

The skill should produce:

- A category label such as landscape / architecture, street / documentary, or still life / product.
- A problem diagnosis for exposure, white balance, contrast, saturation, sharpness, and noise when relevant.
- Lightroom parameters.
- Capture One parameters.
- A short editorial-style visual direction.

## Recommended Workflow

1. Upload or provide the source image.
2. Run the skill to generate the diagnosis and retouching settings.
3. Apply the parameters in your editing software.
4. Use the visual direction to guide further creative refinement.

## Notes

This project is designed to stay structured, concise, and maintainable. The goal is not to replace artistic judgment, but to make the workflow easier to standardize and share.
